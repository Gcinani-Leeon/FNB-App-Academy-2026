# Arithmetic Operations and Type Casting

### Welcome to Unit 3!

This unit will introduce you to the core mathematical operations that form the basis of calculations in Python programs. You will develop an understanding of how Python performs arithmetic and evaluates numerical expressions. The unit also explores how different data types interact and why converting between them is essential when working with user input. In addition, you will use built-in functions to manipulate and refine numeric values. Together, these concepts will equip you to create programs that perform accurate calculations and process numerical data effectively.

## Learning objectives
By the end of this unit, you should be able to:
- Perform arithmetic using Python’s seven operators: +, -, *, /, //, %, **
- Explain the difference between integer division (//) and regular division (/)
- Convert between data types using int(), float(), and str()
- Apply round() and abs() to numeric values
- Build a multi-function calculator using arithmetic and type casting
