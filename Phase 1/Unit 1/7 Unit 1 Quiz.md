### What is a dictionary?
In Python, a dictionary (dict) is a way to store data in key-value pairs. It's exactly like a real-world language dictionary: you look up a "word" (the key) to find its "definition" (the value).

### ​Why is it a "Lookup table"? 
Because you use the key to instantly "look up" the matching value.

### Question 3/10
| 3. Which file name would VS Code recognize as a Python script? |
| --- |
| program.txt |
| program.docx |
| program.py ✅️ |
| program.exe |

​.py is the standard file extension for Python scripts. When VS Code sees this extension, it immediately knows to turn on Python features like syntax highlighting, auto-completion, and that handy run button.

​What the others are:
- ​.txt: A plain text file (no code-running powers here).
- ​.docx: A Microsoft Word document.
- ​.exe: A Windows executable application.
