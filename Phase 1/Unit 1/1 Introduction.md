Welcome to Unit 1!
This unit will introduce you to the foundations of Python programming and its role in solving real-world problems across a range of industries. You will become familiar with the tools needed to write and run Python programs effectively. The unit establishes the basic building blocks of programming by exploring how simple scripts are created and executed. It also develops your understanding of how Python stores and represents different types of information. Together, these concepts provide the foundation for all the programming skills developed throughout the course.

Learning objectives
By the end of this unit, you should be able to:
• Explain what Python is and describe three real-world applications of the language
• Set up a working Python development environment on their own device
• Write and run a basic Python script using print() and input()
• Declare variables using the correct data types: str, int, float, and bool
• Use type() to inspect the data type of any value
