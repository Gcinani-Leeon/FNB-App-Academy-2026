# Conditional Logic and Decision Making
This unit will introduce you to the principles of decision-making in Python, allowing your programs to respond differently based on specific conditions. You will learn how logical comparisons guide the flow of program execution. The unit also explores how multiple conditions can be combined to create more flexible and meaningful outcomes. In addition, you will discover how Python evaluates membership and interprets different values in conditional statements. Together, these concepts will help you build interactive programs that make intelligent decisions based on user input and data.

## Learning objectives
By the end of this unit, you should be able to:
- Write if, elif, and else statements to control program flow based on conditions
- Use comparison operators (==, !=, <, >, <=, >=) correctly in conditions
- Combine conditions using logical operators: and, or, not
- Use the in keyword to check membership in a list or string
- Nest conditional statements where appropriate and explain when to use elif vs separate if
