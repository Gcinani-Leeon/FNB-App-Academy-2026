Before you begin writing Python code, you need to set up the tools that developers use to create and run programs.
In this unit, you will install:
◦ Python
◦ Visual Studio Code (VS Code)
◦ Python extension for VS Code
These tools will provide the environment needed to write, edit, and execute Python programs throughout the course.

Follow the video lesson carefully and complete each installation step as demonstrated. Once you have successfully installed all required software, verify that Python is working correctly and that VS Code can recognise and run Python files. Taking the time to set up your environment correctly will ensure a smoother learning experience in the units that follow.

By the end of this unit, you should have:
• Python installed on your computer.
• Visual Studio Code (VS Code) installed.
• The Python extension installed in VS Code.
• A working Python development environment ready for coding.
Important: Ensure all installations are completed successfully before moving on to Unit 1, as all practical tasks will require these tools.
