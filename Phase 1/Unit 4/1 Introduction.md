# Introduction to Storage and Access

This unit will introduce you to Python’s core data structures for organising and managing collections of information. You will explore how different structures store, retrieve, and organise data to suit different programming needs. The unit also develops your understanding of mutable and immutable data and when each is appropriate. In addition, you will learn how to process structured data efficiently using iteration. Together, these concepts provide the foundation for working with real-world datasets and building more sophisticated Python applications.

### Learning objectives
By the end of this unit, you should be able to:

• Create and manipulate lists using indexing, append, remove, insert, and slicing

• Create and use dictionaries with key-value pairs using .get(), .keys(), .values(), and .items()

• Explain the difference between mutable and immutable data types

• Build a list of dictionaries to represent structured collections of real-world data

• Iterate over lists and dictionaries using for loops
